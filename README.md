# 🏥 Clinical NLP Summarizer

An AI-powered clinical intelligence system that analyzes patient medical records using NLP.

## Features
- 📄 OCR extraction from images and PDFs
- 🧠 AI summarization of medical text
- 🔍 Biomedical Named Entity Recognition (NER)
- ⚠️ Automated Risk Triage (Critical / Urgent / Routine)
- 🤖 Clinical Q&A powered by transformer models
- 📕 PDF report generation
